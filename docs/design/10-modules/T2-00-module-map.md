## 1. 本册定位与写法

- **粒度**：伪代码 + 时序图级（可据此直接开发）。
- **模板**（每模块统一）：概述与定位 → 职责边界 → 接口契约 → 数据模型 → 关键算法（伪代码）→ 错误处理 → **测试设计** → 与现有代码的差量 → 开放问题。
- **测试设计内置**：每模块文档自带该模块的功能测试设计（黄金样本、边界、回归、覆盖目标）；跨模块的集成/系统测试留第三册 T3。
